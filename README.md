# Task Tracker (Менеджер задач)

[![Maintainability](https://qlty.sh/gh/Nizom56/projects/react-task-tracker/maintainability.svg)](https://qlty.sh/gh/Nizom56/projects/react-task-tracker)

## Описание проекта

Интерактивное веб-приложение для управления списком задач. Проект предоставляет пользователю возможность добавлять новые задачи, изменять их статус на «выполнено» и удалять неактуальные записи. За счет интеграции с облачной базой данных все действия синхронизируются и сохраняются в реальном времени без необходимости перезагрузки страницы.

## Стек технологий

* Frontend: React
* База данных: Firebase (Cloud Firestore)
* Деплой: Netlify
